# Document Q&A App

A simple AI-based web app where users can:

- Upload a PDF or text file
- Ask questions about the uploaded document
- Receive AI-generated answers based on the document content

## Tech Stack

- Python
- Flask
- PyPDF2
- Google Gemini API

## Features

- Upload PDF or TXT files
- Extract text from the uploaded file
- Ask questions about the document
- Get answers generated by Gemini
- Simple and beginner-friendly interface

## Project Workflow

1. User opens the web app in the browser.
2. User uploads a PDF or text file.
3. The backend extracts text from the uploaded file.
4. The user types a question about the document.
5. The extracted text and question are sent to Gemini.
6. Gemini generates an answer based on the document content.
7. The answer is shown in the chat interface.

## Setup Instructions

### 1. Install Python
Make sure Python is installed on your computer.

### 2. Install dependencies
Run this command in the project folder:

```bash
pip install -r requirements.txt
```

### 3. Set your Gemini API key

**Windows**
```cmd
set GEMINI_API_KEY=your_api_key_here
```

**Mac/Linux**
```bash
export GEMINI_API_KEY=your_api_key_here
```

### 4. Run the app
```bash
python app.py
```

### 5. Open in browser
Go to:

```text
http://127.0.0.1:5000
```

## How to Use

1. Open the app in your browser.
2. Upload a PDF or text file.
3. Type a question.
4. Click Ask.
5. Read the AI-generated answer.
